# Copyright Headers

## C++

```cpp
/**
 * @file path/under/src/boyle/thing.hpp
 * @author Author Name (author@example.com)
 * @brief Short description.
 * @version 0.1
 * @date YYYY-MM-DD
 *
 * @copyright Copyright (c) YYYY Boyle Development Team
 *            All rights reserved.
 *
 */
```

Replace `YYYY` with the appropriate year.

## Python

```python
# Copyright (c) YYYY Boyle Development Team. All rights reserved.
```

## Notes

- CMake files generally omit this block.
- Cursor `.mdc` rules keep YAML frontmatter on line 1.
