## 1. 概述：什么是 Skill

**Skill** 是一份 Markdown 参考指南，教 AI Agent 如何执行特定任务。它不是一次性的对话历史，而是**可复用的流程定义**——涵盖代码审查标准、告警添加流程、提交消息格式等专业化工作流。

**Skill 的本质**：

| Skill 是                         | Skill 不是                         |
| -------------------------------- | ---------------------------------- |
| 可复用的技术、模式、工具参考     | 关于"我曾经如何解决某个问题"的叙事 |
| 面向 Agent 的结构化操作指南      | 面向人类的教程或文档               |
| 包含判断规则和决策逻辑的流程定义 | 简单的代码片段集合                 |
| 上下文窗口中的"特化知识注入"     | 通用知识的重复                     |

**核心前提**：Agent 本身已经非常聪明。Skill 的价值在于提供 Agent 不具备的**领域特定知识**和**项目特定约定**。
